DOM avançado

Manipulação de DOM

É importante criar os elementos manualmente por JavaScript porque se apenas modificarmos o elemento, todo o elemento será renderizado fazendo com que perca processamento

Para garanti que a página não seja re-renderizada completamente ao adicionar um novo elemento ao DOM, usamos o createElement para criar um novo elemento e o appendChild para adicioná-lo

Criação de elementos

Estrutura para criação de um elemento: document.createElement(ELEMENTO)

Ao criar um novo elemento no JS usando document.createElement o método retornará um objeto Element que representa o novo elemento criado

Inserção de elementos via JS

A forma mais eficiente de adicionar um novo elemento a uma div sem re-renderizar todo o seu conteúdo é por usar o createElement para criar o elemento manualmente e appendChild para inseri-lo na div

Estrutura para inserção de um elemento: LOCAL.appendChild(ELEMENTO_CRIADO)

Atributos e propriedades dos elementos

Estrutura para adicionar um atributo específico em um elemento HTML: ELEMENTO.setAttribute('atributo', 'valor')

Método para adicionar uma nova classe a um elemento HTML: ELEMENTO.classList.add('NOVA-CLASSE')

Estrutura para modificar o estilo inline de um elemento HTML: ELEMENTO.style.fontSize = '20px'

Para remover um atributo de um elemento HTML usando JS: ELEMENTO.removeAttribute('ATRIBUTO')

A diferença entre innerHTML e innerText é que o innerHTML retorna o texto com formatação HTML enquanto o innerText retorna apena o texto

Para obter o valor de um atributo de um elemento HTML usamos a estrutura: ELEMENTO.getAttribute('ATRIBUTO')

Eventos de teclado (key up/down) e identificação de teclas

Ao adicionar um event listener diretamente no HTML poderemos obter detalhes sobre o evento mas é necessário adicionar o atributo 'event' no manipulador de eventos

A propriedade event.code retorna o código físico da tecla pressionada quando um tecla é pressionada

A principal diferença entre as propriedades event.code e event.key em eventos de teclado é que event.key retorna o valor digitado e o event.code retorna o código físico da tecla pressionada

Ao adicionar um event listener diretamente no HTML obteremos detalhes do evento por adicionar o atributo 'event' no manipulador de eventos

O 'onKeyDown' é mais útil que o 'onKeyUp' quando fazemos ações que devem ocorrer continuamente enquanto a tecla é mantida pressionada, como mover um objeto na tela

Evento avançados: Eventos de mouse

mousedown - Evento de clique de mouse quando segura o botão

mouseup - Evento de clique de mouse quando solta o botão

mouseenter - Evento para quando o ponteiro do mouse entrar na área do elemento

mouseleave - Evento para quando o ponteiro do mouse sair da área do elemento

mousemove - Evento de mouse que é disparado ao mover o mouse

Os eventos de mouse podem ser colocados em qualquer elemento html

contextmenu - Evento de clique com o botão direito do mouse

Ao receber um evento como parametro da função, podemos alterar o comportamento padrão com o 'preventDefault'

Exemplo de uso:

event.preventDefault();
