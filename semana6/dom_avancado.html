DOM avançado

Manipulação de DOM

É importante criar os elementos manualmente por JavaScript porque se apenas modificarmos o elemento, todo o elemento será renderizado fazendo com que perca processamento

Para garanti que a página não seja re-renderizada completamente ao adicionar um novo elemento ao DOM, usamos o createElement para criar um novo elemento e o appendChild para adicioná-lo

Criação de elementos

Estrutura para criação de um elemento: document.createElement(ELEMENTO)

Ao criar um novo elemento no JS usando document.createElement o método retornará um objeto Element que representa o novo elemento criado

Inserção de elementos via JS

A forma mais eficiente de adicionar um novo elemento a uma div sem re-renderizar todo o seu conteúdo é por usar o createElement para criar o elemento manualmente e appendChild para inseri-lo na div

Estrutura para inserção de um elemento: LOCAL.appendChild(ELEMENTO_CRIADO)
